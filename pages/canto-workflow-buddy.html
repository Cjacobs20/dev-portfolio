Canto Workflow Buddy v3

Canto Workflow Buddy v3 is a Windows desktop utility built for a BD Canto flow cytometer running on an airgapped lab network. It wraps the instrument vendor's inbound/outbound batch scripts into a single control panel, making lab workflows easier and more reliable.

The app was designed to work completely offline, with no installers or package managers required. It uses PowerShell + WPF for the UI, a borderless Edge app for animation, and simple file-based IPC to communicate between the UI and the status display.

Key features include one-click job execution, automatic result folder monitoring, and a playful status character that reacts to idle, running, success, and failure states.

  • Environment: Windows 10/11, PowerShell 5.1, Microsoft Edge Chromium
  • Tech: PowerShell, WPF/XAML, Three.js, Edge app mode
  • Constraint: Fully offline, airgapped lab system with no external dependencies

This project is a strong portfolio entry because it demonstrates real-world healthcare automation with constrained infrastructure, creative problem solving, and a polished UX built for a locked-down instrument PC.
